vue-web-terminal

一个功能强大的命令式网页仿真终端插件

快速上手在线体验

命令式终端

命令式仿真终端，拥有终端的拖拽、缩放、光标切换、历史命令切换等常规功能，非常适合用于制作 Redis、MySQL、ETCD 等客户端工具。

功能强大

插件默认提供了普通文本、html渲染、Json、表格、代码、Ansi颜色控制码等格式的显示功能，还支持用户问答、动画渲染、在线编辑等高级功能。

主题

插件默认提供暗和亮两种主题，你也可以根据自己的喜好自定义主题，背景、光标、字体等都可以修改成你想要的颜色。

API

插件提供了丰富的Js API，为你模拟非用户行为提供解决方案。

高度自定义

终端显示样式由前端控制，内部提供了丰富的显示样式，也提供了多个 slot，让你可以自定义想显示的任意内容。

Vue2 & Vue3

为了兼容更多的应用场景，插件同时支持了Vue2 和 Vue3！
